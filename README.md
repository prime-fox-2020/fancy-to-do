# fancy-to-do
Fancy To Do App is an SPA to manage your To Do list. This app use : 
* express
* jquery
* ajax

## REST endpoint
- [POST/ register]
- [POST/ login]
- [POST/ google-login]
- [GET/ todos]
- [GET/ todos/:id]
- [POST/ todos]
- [PUT/ todos/:id]
- [DELETE/ todos/:id]
- [POST/ todos/calendar]


## POST/register
register your email and password

### Request Body
```json
{
	"email": "user@email.com",
	"password": "yourPassword"
}
```

### Response

#### 200
```json
{
    "id": 1,
    "email": "user@email.com",
    "password": "&T*UIUGD&^*%*@U877&^$%^^#*bdlsjfbvmjsgf$*6"
}
```

#### 400
```json
[
    {
        "message": "Email Already Exist || validation error"
    }
]
```

#### 500
```json
[
    {
        "message": "internal server error"
    }
]
```


## POST/login
login to your account

### Request Body
```json
{
	"email": "user@email.com",
	"password": "yourPassword"
}
```

### Response

#### 200
```json
{
    "access_token": "random string token"
}
```

#### 400
```json
[
    {
        "message": "wrong email || wrong password"
    }
]
```

#### 500
```json
[
    {
        "message": "internal server error"
    }
]
```

## POST/todos

### Request Headers
```

}
```

```json

	"title": "Todo name",
	"description": "Todo Description",
	"status": "Completed / Not Completed",
	"due_date": "2020-02-02"
}
```

```json
      "description": "<todo description>",
      "status": "<todo status>",
      "due_date": "<todo due date>",
```

